# **App Name**: LearnVerse AI

## Core Features:

- User Authentication: Secure registration/login using Firebase Authentication.
- Personalized Dashboard: Displays user's info, progress, and provides navigation.
- Adaptive Quiz System: Fetches quizzes, adjusts difficulty dynamically, and stores results as per user qualification and education levels (1st to 10th, college, undergraduate and their sub domains , etc)
- AI Tutor Chatbot: Allows students to ask questions and receive AI-generated assistance; tool will assess context and incorporate external information as needed.
- Leaderboard & Badges: Global leaderboard with real-time updates and achievement badges.
- AI Recommendations: Suggests next topics/levels based on quiz performance. Education level dropdown to filter for (1st - 10th, college, university, others).
- Gamified Progress Tracking: Visualize learning progress using progress bars, streak counters and toast notifications upon achievement.

## Style Guidelines:

- Primary color: Soft blue (#8CCDEB) to evoke a sense of calm and focus.
- Background color: Light grey (#FBF5E5) for a clean and neutral backdrop.
- Accent color: Muted teal (#EFB6C8) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font with a modern, objective feel.
- Consistent use of clear and simple icons related to education and achievements.
- Modern, responsive design using Bootstrap/Tailwind CSS. Loading animations during data fetching.
- Subtle animations for toast notifications, badge reveals, and quiz transitions.
